select : 목록중에 선택할수 있도록 해주는 박스를 만들어주는 태그
multiple : 시프트로 여러개 선택, size : 한번에 노출되는 항목의 수, required : 선택필수, disabled : 선택불가
option : select에서 선택할수 있는 항목들
selected : 처음부터 선택되어있는것
optgroup : option들을 묶어서 옵션의 그룹을 나눌 수 있음


fieldset, legend : fieldset으로 form 관련 요소들을 묶고 legend로 그룹의 제목을 제공
사용기술


textarea : 줄글로 작성할 수 있는 텍스트 작성 폼
cols : 너비(기본20), rows : 줄수, maxlength : 최대길이, minlength : 최소길이, placeholder : 도움말
