flex의 부모요소를 flex-container 자식요소를 flex-item 이라고 함.
container안에 item을 1차원적으로 잘 배치하는 방법

flex-direction
row : 주축을 가로로
column : 주축을 세로로

justify-content
flex-start : 정렬을 처음부터
flex-end : 정렬을 끝에서부터
center : 중간정렬
space-between : 배분정렬
space-around : 각각 요소의 축방향 넓이가 일정하게(여백 안겹치게)
space-evenly : 각각 요소의 축방향 넓이가 일정하게(여백 겹치게)

align-items : 교차축 기준 정렬을 어느방향으로 할지
align-content : 교차축의 아이템이 여러줄일때 사용
flex-wrap:wrap 인 상태에서만 사용 가능(한줄배치 or 여러줄배치)

gap : 아이템사이의 간격

flex-grow : 컨테이넡 내부에서 할당하는 공간의 비율
1이상을 주면 가능한한 늘려줌 0을주면 기본값
몇몇개에 2를주고 나머지에 1을 주면 2:1비율로 공간을 할당받음

flex-shrink : 아이템의 크기를 반응형에서 고정(설정한 width값 이상으로 유지)하거나 더 줄어들게 만들어 줌. 1 or 0

align-self : 개별적으로 위치값 조절 가능
