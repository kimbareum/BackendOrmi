table : 감싸는 요소
caption : 제목이나 설명, table의 첫번째 자식으로 사용해야 함
tr,th,td : table row(행), table header(행열의 제목셀, 굵은글씨로 나옴), table data(셀내용,열)
thead, tbody, tfoot : 머리글 본문 바닥글. 구역을 나누는 요소. 선택적으로 사용. 가독성높히는 효과
colspan, rowspan : 열병합, 행병합
colgroup : 테이블 열 그룹을 만들고 싶을 때. 간단한 css 사용 가능
